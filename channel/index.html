    GraphqlChannel

    The GraphqlChannel class is a Ruby on Rails Action Cable channel that allows the client to send GraphQL queries and mutations over a websocket connection.

    The subscribed method is called when the client subscribes to the channel, and it initializes an instance variable @subscription_ids which will store any active GraphQL subscriptions.

    The execute method is called when the client sends a GraphQL query or mutation. It takes in a data object which contains the GraphQL query string, variables, and operation name. It then constructs a context object which includes the current user, the channel, and potentially the current branch ID and current user ID if they are present. The context is passed to the GraphQL schema's execute method along with the query, variables, and operation name. The result of the execute method is then transmitted back to the client.

    The unsubscribed method is called when the client unsubscribes from the channel. It iterates through the @subscription_ids and removes any active subscriptions.

    The ensure_hash method is a private helper method that takes in a query variables object and ensures that it is returned as a hash. If the object is blank, an empty hash is returned. If the object is a string, it is parsed as JSON and returned as a hash. If the object is not a string, it is returned as is.

    ImportChannel

    The ImportChannel class is a Ruby on Rails Action Cable channel that allows the client to receive updates on an import process.

    The subscribed method is called when the client subscribes to the channel, and it starts streaming from the import_channel stream.

    The unsubscribed method is called when the client unsubscribes from the channel. It stops all streams.

    This channel can be used to send updates to the client on the progress of an import process. The client can subscribe to the channel and listen for updates on the import_channel stream. When the import is complete or if the client needs to unsubscribe, the unsubscribed method can be called to clean up and stop the streams.
